# Dainik's Personal Portfolio

Welcome to the repository for my personal portfolio website! This project was built to showcase my knowledge of frontend web development fundamentals, specifically focusing on semantic HTML, modern CSS layouts (Flexbox and Grid), and interactive JavaScript. 

It serves as a central hub for my background as an MScIT student and my technical projects in DevOps, Kubernetes orchestration, and cloud architecture.


## ✨ Features
* **Responsive Design:** Fluid layout that adapts seamlessly to desktop, tablet, and mobile screens.
* **Modern CSS Layouts:** Utilizes **Flexbox** for a clean, adaptable navigation bar and **CSS Grid** for the project showcase section.
* **Interactive UI:** Includes a functional contact form with vanilla JavaScript event handling to prevent default reloading and display dynamic user feedback.
* **Semantic HTML5:** Structured with accessibility and best practices in mind.

## 🛠️ Built With
* **HTML5:** For structural elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
* **CSS3:** Custom styling, variables for theming, and responsive media queries.
* **JavaScript (ES6):** DOM manipulation and event listening for the contact form.
